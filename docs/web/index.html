❓ Como Usar

            

                
1. Inicie o Servidor API

                
Antes de usar a interface web, inicie o servidor:

                cd /Users/rodrigorondon/BCP_CIT_ITAU && python3 run_api_server.py
                
                
2. Escreva sua História

                
Cole ou digite sua história de usuário no formato Markdown. Inclua:

                
    
                    
  • Descrição clara (Como/Quero/Para)
    • 
                    
  • Critérios de aceitação
    • 
                    
  • Regras de negócio
    • 
                

                
                
3. Calcule o BCP

                
Clique em "Calcular BCP" e aguarde a análise.

                
                
4. Analise os Resultados

                
Veja o total de pontos e a distribuição por componente.
